- Usuários
	- Cadastro de usuários

- Plano de contas
- Contas
	- Cadastro de contas
	- Buscar contas pelo seu numero
	- Buscar contas por usuário cadastrado
	- Remoção de conta
- Lançamentos
	- Cadastro de Lançamentos
	- Busca de Lançamentos por ID
